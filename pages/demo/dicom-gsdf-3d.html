DICOM GSDF 3D Model Demo

這個互動頁把數位顯示階調 (DDL)、模擬亮度與 JND 感知索引放在同一個 3D 空間中。 因為瀏覽器無法得知訪客螢幕的實際亮度，Lmin、Lmax 與 gamma 都是用來繪圖的假設值。 它用來輔助理解本專案的 GSDF-inspired transfer table，不代表醫療顯示器校正、DICOM 合規驗證、 HDR 量測或診斷用途。

模型參數

GSDF path: code value 會沿著 JND 空間均勻前進，讓灰階差異更接近人眼可感知的等距。

Gamma path: 一般 gamma baseline 在 JND 軸上的步距不均，暗部或亮部可能比較容易擠在一起。

PQ path: HDR PQ 也採用感知導向設計；此處只作為參考曲線，不等於本 extension 的處理流程。

Simulation boundary: 所有亮度數值只用於模型繪圖；本頁沒有量測或讀取訪客螢幕的實際亮度。

本 demo 會從 CDN 載入 Plotly.js。若網路或內容阻擋器封鎖 CDN，3D 圖表可能無法顯示。
