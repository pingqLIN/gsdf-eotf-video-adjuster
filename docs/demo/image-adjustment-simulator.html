Image Adjustment Simulator

這個頁面用樣本影像與 canvas 模擬本擴充的調整效果。因為網頁無法知道訪客螢幕實際亮度， 所有 Lmax、gamma 與 filter amount 都是相對模擬參數，而不是螢幕量測或校正結果。

Simulated controls

Pipeline: sample pixel → simulated gamma baseline → simulated GSDF table → blend by filter amount. This mirrors the project concept, but it is not reading your display brightness.

The result is intentionally labeled simulated. Use it to understand the extension's intended viewing-aid behavior, not as proof of display calibration or DICOM conformance.

Why simulation? A normal web page cannot measure the visitor's monitor luminance, ambient light, display EOTF, HDR/SDR state, or browser compositor behavior. This demo therefore draws a controlled relative transformation on the sample image so the effect remains understandable and repeatable.
