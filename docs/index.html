Simulated image result

Fixed frequency charts and before/after samples show the intended shadow, haze, and low-contrast detail rescue behavior.

Assumed luminance model

Because a web page cannot know the visitor's actual screen brightness, Lmax and gamma are simulated assumptions, not measurements.

Project boundary

The demo is educational. The extension is a browser-side viewing aid, not a DICOM calibration or display-conformance tool.
